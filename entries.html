    Starting Static

    Friday, 29-Aug-16

    An important aspect of the progress project is to mark and document transformation stages. I have thus far finished two tutorials from codeschool.com giving intros to HTML+CSS. In addition to those tutorials, I explored all of the preparation material for starting the General Assembly - Web Development Immersive. With the learned structures from these paths, I have got a basic website assembled on github pages, that I will now start filling with small blog entries narrating the technicalities augmented, general explorations and discoveries. My very own captains log.

    I coin this as the 'Starting Static' phase of the project. Because documentation is fundamental to posterity, a blog element to my web development journey has always been a destination. And though recording my blog entries as static pages is far from efficient.. it is quick-n-dirty content creation. I eagerly await the decommission of this blog incarnation into a scale-able dynamic infrastructure. Like a good Phoenix.
